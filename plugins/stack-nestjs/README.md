# stack-nestjs

NestJS backend API plugin for Claude Code.

## Features

- NestJS module architecture
- Guards for authentication/authorization
- Service layer patterns
- Prisma query integration

## Agent

- **api-agent**: Tier 2 agent for backend API development

## Skills

- `nestjs-modules`: Module structure, DI patterns, providers
- `nestjs-guards`: Auth guards, RBAC, permission patterns
- `nestjs-services`: Business logic, transactions, error handling
- `prisma-queries`: Query patterns, filtering, pagination

## Installation

```bash
claude /install wigtn/wigtn-plugins-with-claude-code/plugins/stack-nestjs
```
