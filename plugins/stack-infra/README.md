# stack-infra

Infrastructure and DevOps plugin for Claude Code.

## Features

- Docker containerization
- GitHub Actions CI/CD
- Environment management
- Deployment configurations

## Agent

- **infra-agent**: Tier 3 agent for infrastructure management

## Skills

- `docker-configs`: Dockerfile, Docker Compose patterns
- `github-actions`: CI/CD workflows, automation
- `env-management`: Environment variables, secrets

## Installation

```bash
claude /install wigtn/wigtn-plugins-with-claude-code/plugins/stack-infra
```
