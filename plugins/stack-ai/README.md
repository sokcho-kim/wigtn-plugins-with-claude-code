# stack-ai

AI backend integration plugin for Claude Code.

## Features

- LLM integration (OpenAI, Anthropic)
- Vector database patterns (Pinecone, Qdrant, Chroma)
- RAG (Retrieval Augmented Generation) pipelines
- Streaming responses (SSE, WebSocket)

## Agent

- **ai-agent**: Tier 2 agent for AI service integration

## Skills

- `llm-integration`: LLM API patterns, prompt management
- `vector-db`: Embedding storage, similarity search
- `ai-patterns`: RAG, chunking, context management
- `streaming-responses`: SSE/WebSocket streaming

## Installation

```bash
claude /install wigtn/wigtn-plugins-with-claude-code/plugins/stack-ai
```
