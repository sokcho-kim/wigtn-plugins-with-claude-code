# stack-nextjs

Next.js frontend web application plugin for Claude Code.

## Features

- Next.js App Router patterns
- React Server/Client Components
- Custom hooks and state management
- Server Actions for mutations
- Tailwind CSS styling

## Agent

- **webapp-agent**: Tier 2 agent for frontend development

## Skills

- `nextjs-routing`: App Router, layouts, middleware
- `react-components`: Component patterns, props typing
- `react-hooks`: Custom hooks, state management
- `server-actions`: Form handling, mutations
- `tailwind-ui`: Styling patterns, responsive design

## Installation

```bash
claude /install wigtn/wigtn-plugins-with-claude-code/plugins/stack-nextjs
```
