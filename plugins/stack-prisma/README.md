# stack-prisma

Prisma database layer plugin for Claude Code.

## Features

- Database schema design with Prisma ORM
- Migration management and strategies
- Relation patterns (1:1, 1:N, N:M, self-relations)
- Index optimization for query performance

## Agent

- **datamodel-agent**: Tier 1 agent for schema and migration management

## Skills

- `prisma-schema-design`: Model definitions, naming conventions, boilerplates
- `prisma-migrations`: Safe/dangerous patterns, rollback strategies
- `prisma-relations`: Relation setup patterns
- `prisma-indexes`: Index optimization patterns

## Installation

```bash
claude /install wigtn/wigtn-plugins-with-claude-code/plugins/stack-prisma
```
