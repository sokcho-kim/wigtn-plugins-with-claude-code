# stack-admin

Admin dashboard application plugin for Claude Code.

## Features

- Data tables with TanStack Table
- Charts with Recharts
- Admin layouts and navigation
- CRUD interfaces

## Agent

- **admin-agent**: Tier 2 agent for admin dashboard development

## Skills

- `data-tables`: Sorting, filtering, pagination patterns
- `charts`: Dashboard charts, data visualization
- `admin-layouts`: Sidebar, header, navigation

## Installation

```bash
claude /install wigtn/wigtn-plugins-with-claude-code/plugins/stack-admin
```
